# PLANTODO
할 일 관리 웹 페이지

## Environments
- Language : java ("11.0.11" 2021-04-20 LTS)
- IDE : IntelliJ Ultimate 2021.2
- Framework : SpringBoot 2.5.6
- Database : MariaDB 10.11 (java-client 2.7.2)
- ORM : JPA (hibernate)

## Features
- 회원가입, 로그인
- 일정 관리
- 할 일 관리
- 메모 관리
- 푸시 알림
